# wechatselectall

## Description

A tweak that makes u can at everyone in the group without administration in WeChat.

## Install

```
cd wechatselectall
```

add ur device IP into Makefile

```
vi Makefile
THEOS_DEVICE_IP = XXX.XXX.XXX.XXX
```

then make and install

```
make package install
```
